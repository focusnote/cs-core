# CS Core - 核心理论

计算机核心理论知识库，包含数据库原理、计算机组成原理、操作系统、数据结构与算法、编译原理、计算机网络等核心课程。

## 目录结构

- [database-principles](./database-principles) - 数据库原理
- [computer-organization](./computer-organization) - 计算机组成原理
- [operating-systems](./operating-systems) - 操作系统
- [data-structures-algorithms](./data-structures-algorithms) - 数据结构与算法
- [compiler-principles](./compiler-principles) - 编译原理
- [computer-networks](./computer-networks) - 计算机网络

## 内容概述

### 数据库原理
关系型数据库、SQL、事务处理、索引优化、数据库设计等。

### 计算机组成原理
CPU架构、存储器系统、指令集、总线、I/O系统等。

### 操作系统
进程管理、内存管理、文件系统、并发控制、死锁处理等。

### 数据结构与算法
数组、链表、树、图、排序算法、查找算法、动态规划等。

### 编译原理
词法分析、语法分析、语义分析、代码优化、代码生成等。

### 计算机网络
TCP/IP协议、网络分层、路由算法、网络安全、分布式系统等。
